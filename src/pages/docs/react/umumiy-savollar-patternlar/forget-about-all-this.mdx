## Bularni barchasini "Unuting"

**React Forget** — bu React ilovalarida memoizatsiyani avtomatlashtirishga qaratilgan yangi vositalar to'plami bo'lib, ehtimol, `useMemo` va `useCallback` kabi hook’larni ortiqcha qilish imkoniyatiga ega. Memoizatsiyani avtomatik ravishda boshqarish orqali *React Forget* komponentlarning qayta render qilishini optimallashtiradi, foydalanuvchi tajribasini (UX) va dasturchi tajribasini (DX) yaxshilaydi. Ushbu avtomatlashtirish React’ning qayta render qilish xulq-atvorini obyekt identifikatori o'zgarishlaridan semantik qiymat o'zgarishlariga o'tkazadi, chuqur taqqoslashlarsiz, shunday qilib, ishlash samaradorligini oshiradi.

2021-yilda React Conf’da taqdim etilgan *React Forget* hozirda production’da umuman mavjud emas, lekin Meta’da Facebook, Instagram va boshqa platformalarda ishlab chiqarishda foydalanilmoqda va ichki ko'rsatkichlarga ko'ra "kutilgan natijalardan-da oshib ketdi".
